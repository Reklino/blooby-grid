A basic static grid layout with multiple breakpoints.

Basic Static Grid

The container property sets the maximum width of the grid. It should be set to the width of your content. 


bloobyGrid.container = 960,
bloobyGrid.containerPosition = 'center',
bloobyGrid.breaks = [
    {
        point : "(min-width: 800px)",
        columns : 12,
        gutters : .33,
        baseLineHeight : 24
    },
    {
        point : "(min-width: 500px)",
        columns : 6,
        gutters : .16,
        baseLineHeight : 28
    },
    {
        point : "(min-width: 300px)",
        columns : 3,
        gutters : .08,
        baseLineHeight : 24
    }
];
bloobyGrid.gridInit();
