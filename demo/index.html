Blooby
Grid

An itty bitty tool for visualizing fluid grids during development.

Define the container

If the container's max width is undefined, it will be set to the full width of the the page. The position defaults to center but accepts 'left', 'center', or 'right'; 

bloobyGrid.maxWidth = 960,
bloobyGrid.position = 'center';

Add a breakpoint or two

Each breakpoint object has 4 required properties:

  • The 'point' property accepts any valid css media query.
  • The 'columns' property should be set to however many columns are active at this breakpoint.
  • The 'gutters' determine the space between the columns in percentage.
  • The 'baseLineHeight' sets the base lineheight in pixels.

bloobyGrid.breaks = [
    {
        point : "(min-width: 800px)",
        columns : 12,
        gutters : .33,
        baseLineHeight : 24
    },
    {
        point : "(min-width: 300px)",
        columns : 3,
        gutters : .33,
        baseLineHeight : 24
    }
];
